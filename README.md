ThanksPress
===========

Let's be thankful.

Installation
------------

...make sure you are insite thankspress main directory to follow these commands
- pip install virtualenv
- virtualenv venv
- source venv/bin/activate
- pip install -r requirements.txt
- db_create.py
- db_migrate.py
- run.py
